# 🌧️ 影子互文 · Shadow Intertext

> 给AI一个可以随时换衣服、贴备忘、继承经验的壳，让它每次学新东西的时候，不用再改自己。

[🌐 在线体验](https://你的用户名.github.io/shadow-intertext)

---

## 这是什么

我们教AI新东西的时候，为什么要每次都去改它的"脑子"？

影子互文的想法很简单：**给每个词配一个影子，不让它改自己，只给它换衣服。**

这个影子不是人类的影子（人类的影子是扁的，丢了一个维度）。AI世界里的影子是**高维的**——每个词像一座冰山浮在水面上，水面下是它在无数语境里积累下来的可能性。

当AI遇到一句话，AI的向量和影子的向量同步激活。AI先读影子层，影子层有提示或风险提示——相当于**AI可读的灵活化提示词**。

**但重点是：整个过程，AI自己不需要被改写。**

## 交互演示

- 输入框中写一句关于"雨"的感受
- 存入个人记忆库（待审核）
- 模拟审核通过后，碎片飘入影子层
- 随时间慢慢"腐烂"（温和的遗忘，不是抹除）
